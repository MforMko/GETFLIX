myFlix

This web app will allow users to access information about movies, directors and film genres. Users will be able to sign up, update their personal information and create a list of their favourite movies.

Business Logic URL HTTP Method Request Body Data Format Response Body Data Format
Return a list of all movies /movies GET None A JSON object holding data about all the movies
Return data of a single movie by title /movies/[title] GET None A JSON object holding all data about a single movie
Return data (description) about a genre by name (e.g. “Thriller”) /movies/[genre]/[name] GET None A JSON object holding all data about a genre (e.g., “Thriller”)
Return data (bio, birth year, death year) about a director by name /movies/[director]/[name] GET None A JSON object holding all data about a director
