GETFLIX

This web app will allow users to access information about movies, directors and film genres. Users will be able to sign up, update their personal information and create a list of their favourite movies.

API Documentation

Retrieve All Movies

Description: Retrieves all movies.
Endpoint URL: /movies
Method Type: GET
Query Parameters: None
Request Body Data Format: None
Response Body Data Format: Array of JSON objects representing movies.

Retrieve Movie by Title

Description: Retrieves a movie by its title.
Endpoint URL: /movies/:title
Method Type: GET
Query Parameters: ':title'
Request Body Data Format: None
Response Body Data Format: JSON object representing the movie.

Retrieve a Genre Description by Name

Description: Retrieves a genre (description) by name.
Endpoint URL: /movies/:genre/:genreName
Method Type: GET
Query Parameters: ':genre', ':genreName'
Request Body Data Format: None
Response Body Data Format: JSON object representing the movie.

Retrieve Director Information by Name

Description: Retrieves a director's information by name.
Endpoint URL: /movies/director/:directorName
Method Type: GET
Query Parameters: ':director', ':directorName'
Request Body Data Format: None
Response Body Data Format: JSON object representing the movie.

Create New User

Description: Creates a new user with a unique ID.
Endpoint URL: /users
Method Type: POST
Query Parameters: None
Request Body Data Format: JSON object with user information. Example:
{ "name": "John Doe" }
Response Body Data Format: JSON object representing the newly created user.

Update User Information

Description: Updates user information based on provided user ID.
Endpoint URL: /users/:id
Method Type: PUT
Query Parameters: ':id'
Request Body Data Format: JSON object with updated user information. Example:
{ "name": "New Name" }
Response Body Data Format: JSON object representing the updated user.

Add Movie to User's Favourite Movies

Description: Adds a movie to a user's favourite movies list based on provided user ID and movie title.
Endpoint URL: /users/:id/:movieTitle
Method Type: POST
Query Parameters: ':id', ':movieTitle'
Request Body Data Format: None
Response Body Data Format: Text message indicating the movie has been added to the user's array.

Remove a Movie from a User's Favourite Movies

Description: Removes a movie from a user's favourite movies list based on provided user ID and movie title.
Endpoint URL: /users/:id/:movieTitle
Method Type: DELETE
Query Parameters: ':id', ':movieTitle'
Request Body Data Format: None
Response Body Data Format: Text message indicating the movie has been added to the user's array.

Remove a User

Description: Allows a user's to de-register from GETFLIX.
Endpoint URL: /users/:id
Method Type: DELETE
Query Parameters: ':id'
Request Body Data Format: None
Response Body Data Format: Text message indicating the movie has been added to the user's array.
